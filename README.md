# Rhythm-Reaction-Trainer

### Before you start
1. You can't download the zip file, as it is not built as an app but rather a web.
##
### How to run (local)
1. Create a copy of the Rhythm Reaction Trainer.
2. Open terminal and type the commands in order:
    - npm install -g serve
    - npm fund (Optional)
    - serve . -l 8000 (it is an L not a one)
3. Once done you will get a notification saying "Your application running on port 8000 is available
4. Select open in browser
5. Type your username and press start
##
# How to play
### Features:

* The log in button:

  * Logging into your account before playing the game, and it saves if you leave the game without logging out.
  * If logged in, the user can see their recorded stats by:

    * clicking on their username next to the settings icon
    * To view detailed stats, click on detailed summary

      * This allows you to filter your summarised stats to however you want.
* The sign up button:

  * Creating an account from scratch, but making a password requires special characters, numbers, Capital Letters, and lowercase letters for security purposes.
* Statistics/Performance section:

  * Your stats are recorded under the gameplay screen for every mode and difficulty you play.
* Settings:

  * If logged in, the settings adds the account options to either reset any modes or all modes at once, delete account, and log out
  * Upload any audio/video file in background music section to play music in background. It will be paused whilst playing any game mode.
  * Background music has the option to loop infinitely unless disabled.
  * You can enable Pattern Guide in the settings.
* All modes have audio, so it is best to use headphones/earphones when playing.



### Game Modes:

* #### Beat Click:

  * Beat click allows you to click to the beat and can help sharpen your reaction timing, you must press the **backspace** button on your keyboard or click the **Stop** button next to the start in order to end the mode.
* #### Key Press:

  * Key press allows you to press a specific letter key once the letter touches the white line at the bottom of the game box (where all the gameplay happens)
  * The keys can be changed in the keybinds section in settings.
  * Key press doesn't require the backspace button since it automatically ends when a single round is complete.
* #### Pattern Memory Mode:

  * Pattern memory mode allows you to mimic the beats given by one of the 7 tiles.

    * Easy mode (noob, ez) Tiles: 1 2 3
    * Medium mode (Veteran, Experienced) Tiles 1 2 3 4 5
    * Hard mode:

      * Expert - Tiles 1 2 3 4 5 6
      * Pro - Tiles 1 2 3 4 5 6 7
  * The easiest difficulty starts at 5 rounds, and each difficulty adds another round depending on the difficulty order (easy - medium - hard)
  * All tile orders are randomised and all tiles can be reused in any order.

### 

### Difficulty:

* All difficulties can alter the speed of any game mode (Noob: Slowest speed rate | Pro: Fastest speed rate).





### Developer Console:

* There is a dev console that can be activated by using a Konami sequence (you can type it anywhere in the browser, you don't have to be playing any mode while doing so)

  * ↑  ↑  ↓  ↓  ←  →  ←  →  R  R  T  -  6  .  0
* In dev console, you can set up the order of the pattern memory mode in any order you want up to the max rounds (10), this means if you use Pi7 in noob difficulty first, you get that chosen tile no matter what difficulty you are. Although the amount of rounds can't be overridden or changed.
* Key Press Speed used for altering the speed of any difficulty when playing the Key Press mode.
* Auto Clicker is used to automate any game mode at any difficulty, the auto clicker has two combo options which are gurrenteed to have if chosen; Perfect combos and Good combos.
* Stats Override is used to alter the statistics of a users performance and scores:

  * Enable score multiplier allows the user to multiply a single score earned via any game mode by any value (No Limit)
  * Enabling Stats Editing allows you to double click on and edit your scores, plays, accuracy, precision, best combos, and high scores.

    * To disable stats, simply uncheck the stats editing feature.
  * Revert stats is used to undo any stats changed to your real stats.
  * WARNING: May cause loss of actual stats and may reset to 0.
* Tile Previews are used to preview the pattern guide for each tile with built-in visual hitboxes.
* Media Previewer allows the user to watch any used media from this game (e.g. gif, mp3, mp4, png)
* Game Controls:

  * Refresh Page is for people who don't know what Ctrl + R does

    * It used to test the dev console still being able to open even after a refresh.
* To close the dev console, retype the Konami sequence.





### Easter eggs (For people who like hidden surprises):

* First Easter Egg:

  1. To get the easter egg, go to the settings (doesn't matter if you are logged in or not)
  2. Then change all key binds to from A S D F to N  E  D  A (will revert back to default key binds after)
  3. press save and THEN cancel
  4. After that hover on a gif on the bottom right corner and type the Konami code T  W  O  F  O  R  T
  * To close the easter egg, just click out of it
* Second Easter Egg (Key Press):

  1. Repeat Steps 1-4 for 1st easter egg
  2. Hover over the gif and type the 3 Konami code to unlock bonus difficulties (Not really difficulties):

     * POOTS
     * HEAVYDANCE
     * HEAVYWEAPONSGUY
  3. Go to Key Press mode
  4. Change the difficulty to what's under pro.
  * WARNING: May experience lag and buffering; restart the game mode (not refresh) to fix the issue (requires good WIFI for better gameplay)
