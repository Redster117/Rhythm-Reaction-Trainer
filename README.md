There are two things to remember:
1. To begin running the game, you mustn't download the zip file, as it is not built as an app but rather a web.
2. (Spoilers) Whenever you create a codespace from my main one, the demo.gif doesn't appear in the docs folder. (I don't know why this happens ¯\\_ _( ͡° ʖ̯ ͡°)_ _/¯ )
You are gonna need to find the demo.gif under .gitignore and place it in the docs folder in explorer. (If this problem happens, then after you resolve it, you shall see why the gif is needed.)

## How to run (local)
1. Create a copy of the Rhythm Reaction Trainer.
2. Open terminal and type the commands in order:
    - npm install -g serve
    - npm fund (Optional)
    - serve . -l 8000 (it is an L not a one)
3. Once done you will get a notification saying "Your application running on port 8000 is available
4. Select open in browser
5. Type your username and press start
